##Auction API (Django REST Framework)
A backend auction system built with Django REST Framework that allows users to create auctions, place bids, and automatically determine winners.
This project demonstrates REST API design, authentication, and business logic handling in Django.

##Features

- User registration & authentication
- Create, update, and delete auctions
- Place bids with validation
- Highest bid tracking
- Auction closing logic
- RESTful API structure
